# Null-Gravity-VR: Розробка VR-симулятора з використанням фізичних моделей нульової гравітації в середовищі Unity

Кваліфікаційна робота на підтвердження ступеня фахового молодшого бакалавра ([ВСП «ППФК НТУ «ХПІ»](http://polytechnic.poltava.ua)). 

**Керівник роботи** – Подунова Анастасія Олександрівна.

---

## Завдання до дипломної роботи
Розроблений програмний продукт має являти собою VR-симулятор у жанрі Sci-Fi з фізикою нульової гравітації. Проєкт передбачає переміщення по космічній станції за допомогою джетпака та інерції, а також бойову взаємодію з дронами. Система має відстежувати рівень здоров’я та паливо гравця і забезпечувати футуристичне оформлення. Метою проєкту є відновлення систем життєзабезпечення станцій (енергоживлення, гравітація, вентиляція).

## Використані технології
* **Unity** — основний ігровий рушій для розробки.
* **C# (.NET)** — реалізація фізики джетпака, систем життєзабезпечення та ШІ дронів.
* **XR Interaction Toolkit** — база для взаємодії в VR та відстеження рухів гравця.
* **Unity Physics (PhysX)** — моделювання інерції та безвагової взаємодії об'єктів.
* **Blender** — моделювання космічної станції та Sci-Fi об'єктів.
* **Git** — система контролю версій.
